🧠 중앙 목표

CSV/XLSX 업로드 → AI 분석 → AutoML → SHAP → PDF/리포트 → DB 저장 → 자동 재학습까지 이어지는 운영형 AI 플랫폼입니다.

🔐 인증/권한

  • Free: 분류 중심
  • Pro: 분류+회귀+SHAP+PDF+배치
  • Enterprise: AutoML+강화학습+Cron관리
  • Admin: 사용자/플랜 관리

🤖 AI 학습

  • 지도학습
  • 비지도학습
  • 준지도학습

  • AutoML 엔진 골격

📊 분석 파이프라인

  • Target 자동 탐지

🗄️ DB/이력

  • MariaDB: 사용자/플랜/이력

  • JSON 로그: 항상 남기는 안전 기록
  • DB 삭제 없음

📂 파일/리포트

  • 중복 파일 감지

  • PDF/TXT 리포트 구조

  • systemd 자동 시작
